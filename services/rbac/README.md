# RBAC Service

Role-Based Access Control and Authentication service.

## 🛡️ Responsibilities

- User registration and login.
- JWT generation and revocation.
- API Key management.
- Admin and Site User role assignments.

## 📍 Endpoints

- `/api/auth/login`
- `/api/auth/register`
- `/api/admin` - Admin management
- `/api/site-users` - Site level user management
